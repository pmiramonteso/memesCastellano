Documentación de la API

Aquí se encuentra la documentación para la API de memes. Explora los detalles de cómo utilizarla y los métodos disponibles.

URL base:

http://localhost:3000/api/memes

Método de autenticación:

Clave API

Formatos de respuesta:

JSON

Versión de la API:

v1

Introducción

Esta API permite acceder a memes categorizados, con categorías como "programadores", "gatos", y más. Puedes consultar memes aleatorios y buscar por categoría.

Autenticación

Para acceder a la API, necesitarás una clave API. Regístrate para obtenerla en /api/generar-api-key.

Métodos de la API

  • auth
    • GET: /api/generar-api-key - Genera una nueva clave API
  • categorias
    • GET: /api/memes/categorias - Obtiene todas las categorías de memes
    • GET: /api/memes/categorias/:categoria - Obtiene memes por categoría
  • demo
    • GET: /api/memes/demo/random - Obtiene un meme aleatorio

Ejemplo completo de solicitud

GET http://localhost:4200/api/memes/categorias/gatos

Headers:

  • x-api-key: tu-clave-api-aqui

Respuesta:

"id": 1, "titulo": "Meme de gatos", "descripcion": "Un gato haciendo algo gracioso", "imagen": "http://localhost:3000/assets/img/gato.jpg"
